COMANDI REPO base
-------------------------------
"git clone" "indirizzo repo" copia la repository in locale
"git init" "nome repo" inizializza una repo locale
"git add" "file da aggiungere" aggiunge un file all' 'indice dei cambiamenti
"git commit" registra i cambimenti, chiedera di effettuare un commento su di essi
"git push" carica i cambiamenti online su git

COMANDI REPO avanzati 
-------------------------------
"git branch"
"git merge"
